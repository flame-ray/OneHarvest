# OneHarvest Wiki

## 简介

OneHarvest 是一个高效采集插件。

支持：

* 连锁砍树
* 连锁挖矿
* 3×3 范围挖掘
* 可调范围
* 可调最大采集数量
* 玩家独立开关
* 高性能优化
* 兼容 Minecraft 1.21.x

---

## 安装方法

### 1. 下载插件

将 `OneHarvest-1.1.0-Paper-1.21.x.jar` 放入服务器 `plugins` 目录。

### 2. 重启服务器

执行 `stop` 后重新启动服务器即可。

## 功能介绍

### 连锁砍树

当玩家破坏原木时会自动搜索相连的原木并破坏整棵树，同时掉落物品。支持橡木、白柄木、云杉木、丛林木、金合沙木、深色橡木、红树木、樱花木、苍白木、下界菌柄等原木类型。

### 连锁挖矿

挖掘矿矿时自动挖掘同矿脉的所有矿矿。支持煤矿矿、铁矿矿、铜矿矿、金矿矿、红石矿矿、青金矿矿、钻石矿矿、绿宝石矿矿、下界金矿矿、下界石英矿矿及深层矿矿。

### 范围挖掘

支持石头、圆石、深板崎、泥土、草方块、砂土、砂石、凝灰岩等方块的范围挖掘。默认 3×3，可设为 1×1、3×3、5×5 或 7×7。

## 指令

### 查看帮助

```
/oh help
```

### 开启或关闭插件

```
/oh toggle
```

### 修改范围

* `3×3`: `/oh range 1`
* `5×5`: `/oh range 2`
* `7×7`: `/oh range 3`

### 修改最大连锁数量

```
/oh max 10000
```

### 重载配置

```
/oh reload
```

## 权限节点

* `oneharvest.use` — 基础使用
* `oneharvest.admin` — 修改设置
* `oneharvest.reload` — 重载配置
* `oneharvest.*` — 全部权限

## 配置文件

`config.yml` 示例：

```yaml
enabled: true

max-chain-break: 10000

vein-mining: true

tree-felling: true

area-mining: true

default-range: 1

drop-items: true
```

### 配置说明

* **enabled** — 插件总开关
* **max-chain-break** — 最大连锁数量，推荐 5000~10000
* **vein-mining** — 矿脉挖掘开关
* **tree-felling** — 连锁砍树开关
* **area-mining** — 范围挖掘开关
* **default-range** — 默认范围 (0=1×1, 1=3×3, 2=5×5, 3=7×7)

## 常见问题

**Q：会掉落经验吗？**

会，经验掉落与原版一致。

**Q：支持精准采集和时运吗？**

支持，附驯效果正常生效。

**Q：会导致服务器占用容量增加吗？**

不会，插件采用限制搜索算法，默认上限 10000 已足够安全。

**Q：需要客户端安装吗？**

不需要，只有服务端安装即可。
